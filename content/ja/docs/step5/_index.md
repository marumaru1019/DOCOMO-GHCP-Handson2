## 🎯 課題の概要

GitHub Copilot のエージェントモードを活用して、簡単なWebアプリケーションを設計から実装まで一貫して開発する課題です。これまで学んだ高度機能（エージェントモード、ツール、`#codebase`など）を総合的に活用し、実務レベルの開発体験を積むことを目的とします。

---

## 📋 課題内容

### 🎯 お題
**GitHub Copilot エージェントモードを使って、以下の要件を満たすWebアプリケーションを開発してください。**

#### 必須要件
1. **フロントエンド技術**を使用（HTML/CSS/JavaScript、React、Vue.js、Angular など）
2. **複数のファイル**で構成される（最低3ファイル以上）
3. awesome-copilot から活用できそうなカスタム指示やチャットモードを1つ以上活用

#### 推奨要件（任意）
- レスポンシブデザイン対応
- 簡単なアニメーションやエフェクト
- ローカルストレージの活用
- 外部APIの利用
- README 等のドキュメントファイルの作成
- 単体テストの実装

### 🚀 アプローチ

#### 1. 企画フェーズ（5分）
- どんなアプリを作るか決める
- 必要な機能を整理する
- 技術スタックを選択する

#### 2. 開発フェーズ（25分）
- エージェントモードで企画の内容を設計に落とし込む
- 機能を段階的に実装
- 動作確認とデバッグ

## 🎨 サンプルアプリケーション例

開発の参考として、以下のようなアプリケーションが考えられます。**これらはあくまで例**なので、独自のアイデアで開発してください。

### 1. 📝 タスク管理アプリ
**機能:** タスクの追加・削除・完了切り替え・絞り込み表示</br>
**技術:** React/Vue.js + CSS + localStorage

### 2. ✂️ じゃんけんゲーム
**機能:** プレイヤー選択・コンピュータ対戦・勝敗判定・スコア表示</br>
**技術:** JavaScript + CSS + HTML

### 3. 🌤️ 天気アプリ
**機能:** 現在地の天気表示・5日間予報・地域検索</br>
**技術:** React + Weather API + レスポンシブデザイン

### 4. 📊 データ可視化ダッシュボード
**機能:** チャート表示・データフィルタリング・エクスポート機能</br>
**技術:** Chart.js + JavaScript + CSV読み込み

### 5. 🎮 ミニゲーム
**機能:** スコア管理・レベルシステム・ハイスコア保存</br>
**技術:** Canvas API + JavaScript + CSS Animation

### 6. 📅 カレンダーアプリ
**機能:** イベント追加・月間表示・リマインダー機能</br>
**技術:** JavaScript + Date API + localStorage

### 7. 💰 家計簿アプリ
**機能:** 収支入力・カテゴリ別集計・グラフ表示</br>
**技術:** React + Chart.js + データ永続化

### 8. 📝 マークダウンエディタ
**機能:** リアルタイムプレビュー・ファイル保存・構文ハイライト</br>
**技術:** JavaScript + marked.js + CodeMirror

### 9. 🔳 QRコード生成ツール
**機能:** URL入力・QRコード生成・画像ダウンロード（PNG/SVG）・履歴保存/再利用</br>
**技術:** JavaScript + QRコード生成ライブラリ（qrcode.js など） + localStorage + CSS

### 10. 🎯 クイズアプリ
**機能:** 問題表示・選択肢・スコア計算・結果画面</br>
**技術:** React/Vue.js + JSON データ + CSS Animation

## ⏰ タイムスケジュール

| 時間 | 活動内容 | 詳細 |
|------|----------|------|
| **30分** | **もくもくタイム** | 個人でアプリケーション開発に集中 |
| **20分** | **情報交換** | 作成したアプリの共有と工夫点の意見交換 |
| **20分** | **全体発表** | 各グループの成果発表と学びの共有 |

### 📢 情報交換のポイント
情報交換タイムでは、以下について共有してください：

- **作成したアプリケーションのデモ**
- **エージェントモードへの指示で工夫した点(カスタムインストラクション等のカスタマイズの工夫も含めて)**
- **うまくいった指示の出し方**
- **つまずいた点と解決方法**
- **予想外の結果や発見**

> **📌 ポイント**
>
> この課題の目的は、**GitHub Copilot エージェントモードを実際の開発に活用する感覚を掴む**ことです。完璧なアプリケーションを作ることよりも、エージェントモードとの協働でどのような開発体験が得られるかを体感してください。
